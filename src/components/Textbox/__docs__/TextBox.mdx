# TextBox

TextBox component with different props.

## Usage

```ts
import { TextBox } from 'lazy-smart-ui-lib';

const Example = () => {
  return (
      <TextBox
        size="small"
        id="test"
        value="test"
      />
  );
};

export default Example;
```

#### Arguments

- **completion** - An array of string for autocompletion
- **custom** - A string that represents an external class to overwrite the style.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **id** - A string that represnt the id of the input
- **onChange** - A function that is called when the input value changed
- **onError** - A function that is called when the input value is in error
- **placeholder** - A string that represents the placeholder
- **readonly** - A boolean to know if the input is readonly or not
- **size** - A string with one of three possible values: "small", "medium", or "large". It defines the size or dimensions of the button.
- **theme** - A string with one of three possible values: "naked", "classic", or "outlined". It defines the style of the button display.
- **validator** - A function to test if the data is correct
- **value** - A string that represents the default value for the input
