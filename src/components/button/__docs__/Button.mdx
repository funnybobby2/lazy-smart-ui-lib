# Button

Button component with different props.

## Usage

```ts
import { Button } from 'lazy-smart-ui-lib';

const Example = () => {
  return (
      <Button
        size="small"
        label="Click me !"
        onClick={()=> console.log("Clicked")}
        disabled
      />
  );
};

export default Example;
```

#### Arguments

- **custom** - A string that represents an external class to overwrite the style.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **label** - A ReactNode, can be a simple string or a component like an svg.
- **onClick** _`() => void`_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
- **size** - A string with one of three possible values: "small", "medium", or "large". It defines the size or dimensions of the button.
- **theme** - A string with one of three possible values: "naked", "classic", or "outlined". It defines the style of the button display.
