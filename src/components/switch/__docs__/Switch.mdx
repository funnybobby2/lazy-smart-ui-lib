# Switch

Switch component with different props.

## Usage

```ts
import { Switch } from 'lazy-smart-ui-lib';

const Example = () => {
  return (
      <Switch
        label="Toggle"
        onClick={()=> console.log("Clicked")}
      />
  );
};

export default Example;
```

#### Arguments

- **custom** - A string that represents an external class to overwrite the style.
- **disabled** - A boolean indicating whether the switch should be disabled or not. When disabled, the switch cannot be clicked or interacted with.
- **label** - A ReactNode, can be a simple string or a component like an svg.
- **labelPosition** - A string to know where the label is positionned.
- **isSimple** - Change the style of the switch button to be very simple
- **onClick** _`() => void`_ - A function that is called when the switch is clicked. It receives a MouseEventHandler for handling the click event on the button element.
